# Site Informativo Minecraft

O site possui objetivo de explicar sobre o jogo Minecraft, sua criação, informações básicas, jogabilidade e novas atualizações.

# Tecnologias Utilizadas:

- Bootstrap
- HTML
- CSS
- PHP
- JSON

# Desenvolvimento:
